接口说明：

ajax方式的API是ajax，另外还有ajaxType、ajaxData、async等三个接口配合使用，分别是ajax的type、data、async。

应用代码：

var pop = saogaUI.ui.pop({
	title:'弹窗的标题',
	height:240,
	width:320,
	ajax:'http://127.0.0.1/saogaUI/doc/tmp/pop.php',
	async:false,
	btns:[
		{
			text:'保存修改',
			closePop:false,
			onclick:function(id){
				alert('保存修改')
				pop.close(id);
			}
		},
		{
			text:'取消',
			onclick:function(){
	
			}
		}
	]
});
