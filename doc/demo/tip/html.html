接口说明：

该接口的是直接在tip容器中插入html，为了兼容原先的代码，还保留该接口，建议使用升级版render接口，更灵活。

应用代码：

var html = '';
html += '<div id="lt-wrap">';
html += '	<div data-message="请速发货1！" class="ui-btn ui-btn-primary buyerMessagePop" >鼠标移动到此<div/>';
html += '<div/>';

saogaUI.ui.tip({
	target:'.buyerMessagePop',
	targetWrap:'#lt-wrap',
	width:100,
	html:html
});

调用效果：

鼠标移动到此
