列统计：

1)、statis

statis接口是一个json格式的数据，display是统计的名称；type是统计的类型，共有sum、avg、min、max四种类型，暂不支持其他复杂形式的统计。

2)、statisWrap

该接口是columns数组里面的数据的一个属性，用来放置statis接口display的名字，不设置将默认第一列。

3)、statisType

该接口是columns数组里面的数据的一个属性，用来指定statis要统计的类型。

行统计：

1)、statis

该接口是columns数组里面的数据的一个属性，若不设置或为false，行统计值为undefined。

2)、statisType

该接口是columns数组里面的数据的一个属性，用来指定statis要统计的类型。

应用代码(列统计)：

var gridCell = saogaUI.ui.grid({
	wrap:'#gridCell',
	pageAjax: {
		url: 'http://127.0.0.1/saogaUI/doc/tmp/grid.php',
		data:'type=1&sex=man',
	},
	statis:[
		{display:'均值',type:'avg'},
		{display:'总计',type:'sum'},
	],
	columns: [
		{ display: '编码.', name: 'id', width: 50, statisWrap:true},
		{ display: '姓名', name: 'name', width: 50},
		{ display: '类别', name: 'type', width: 50},
		{ display: '性别', name: 'sex', width: 50},
		{ display: '电话', name: 'tel', width: 50, statisType:'avg,sum'},
		{ display: 'Email', name: 'email', width: 50},
	]
});
