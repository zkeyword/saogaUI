saogaUI是什么？

saogaUI是基于 jquery1.8.3、less、requirejs、font-awesome-4.2.0 开发的UI类库，开发时请先了解相关的知识，该库支持IE8+浏览器，若要支持IE67的请自行修改代码。

saogaUI包含什么？

grid表格

dialog对话框

pop弹出框

tip提示块

tab标签切换

select2下拉(多选)模拟

drag拖拽组件

calendar日历控件

工具安装

saogaUI是使用Node.js开发，需要先安装Node.js，再通过npm安装相应的编译和打包工具，如果你不用node.js可以选择其他的方式编译less和打包requirejs。

npm install -g

修改config-saogaUI.json中的目录地址，运行 run.bat 就可以监听编译less和打包requirejs。

{
	"dir":"saogaUI"
}

快速入门

引入脚本：

<script src="../build/js/lib/jquery-1.8.3.min.js"></script>
<script data-main="../build/js/common" src="../build/js/lib/require.js"></script>

引入样式：

<link rel="stylesheet" type="text/css" href="../build/css/styles.css">

调用saogaUI：

require(['app/main'], function(saogaUI){
	//开始 console.log(saogaUI)
});
总体结构： 
<!DOCTYPE HTML>
<html lang="en-US">
<head>
	<meta charset="UTF-8">
	<title>saogaUI</title>
	<link rel="stylesheet" type="text/css" href="theme/build/css/styles.css">
	<script src="../build/js/lib/jquery-1.8.3.min.js"></script>
	<script data-main="../build/js/common" src="../build/js/lib/require.js"></script>
</head>
<body>
	<script type="text/javascript">
		require(['app/main'], function(saogaUI){
			//开始 console.log(saogaUI)
		});
	</script>
</body>
</html>
