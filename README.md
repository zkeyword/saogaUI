saogaUI 一个基于jQuery的UI库
=============================


文档未齐



由于个人很喜欢日本的动漫，所以取日语中的そうか的译音saoga为前缀，意义不大只是为了好记。

saogaUI的前身是个人编写的[zUI]https://github.com/zkeyword/zUI ,由于这套UI写的不够满意，所以决定在这个的基础上丰富起来，并且采用LESS和RequireJS进行模块开发，提高了代码的可维护性。

目前，saogaUI提供如下功能：

1、grid 表格

2、dialog 对话框，包含alert、confir、merror、prompt等对话框。

3、pop 弹出框

4、tip 箭头的浮动提示框

5、tab 选项卡

6、selectArea 地区选择

7、calendar 日历组件

8、常用按钮、表单、图标、CSS3圆角、三角、CSS3动画

其中，感谢wangmiaoduan提供支持，saogaUI已经抛弃IE8以下版本，如需支持请移步[zUI]https://github.com/zkeyword/zUI。
